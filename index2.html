📋 조회 탭 이용 방법

1. 상단 검색란에 요청자 이름을 입력하고 "검색" 버튼을 클릭합니다.

2. 해당 이름의 접수 내역이 드롭다운으로 표시됩니다. (최대 10건, 최신순)

3. 접수번호를 선택하면 질문과 답변이 대화 형식으로 표시됩니다.

4. "📥 엑셀 다운로드" 버튼으로 전체 데이터를 다운로드할 수 있습니다.

📌 상태 안내

• 답변 완료 — 담당자가 답변을 등록한 상태입니다.

• 답변 대기 — 아직 답변이 등록되지 않은 상태입니다. 노란색 배경으로 표시됩니다.
